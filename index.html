Hello, my friend! What is a computer?

A computer is an electronic machine which can accept data in a certain form and give the results as information.

Data is fed into the computer memory. Then, when the program is run, the computer performs a set of instructions and processed the data. Finally, we can see the results on the screen or in printed form.

A computer system consists of to parts: HARDWARE(any electronical or machanical part you can see or touch) and SOFTWARE(a set of instructions, which tells the computer what to do).

The CPU(central processing unit) is the most influential component. Its function is to execute program instructions and coordinate the activities ol all the other units.
